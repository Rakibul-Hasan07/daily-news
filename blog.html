1. Difference between var, let and const

Var: Redefine possible and re assign possible. Let: re assign possible but re define not possible and let is a block scope. const: redefine not possible and re assign not possible and const ia a block scope.

2. Difference between arrow and regular function

Arrow functions are best for callbacks or methods like map , reduce , forEach, find and filter. arrow function a shorter syntax and thus requires less code.Regular functions created using function declarations or expressions are constructible and callable.

3. Difference between map, forEach, filter and find

Using map of array. map work process like loop. map passing return value. forEach work like map but forEach can't return value. filter work like loop then return all matched value conditions wise. find return first matched value conditions wise

4. Why we are using template string

Template strings are a powerful feature of modern JavaScript released in ES6. Using template string set value dynamically.It allows us to create strings that are complex and contain dynamic elements.
